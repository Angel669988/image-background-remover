# Image Background Remover - MVP 需求文档

## 1. 产品概述

**产品名称**：图片背景移除器  
**核心功能**：用户上传图片，自动移除背景，返回透明背景的PNG图片  
**目标用户**：电商卖家、设计师、需要透明背景图片的用户  
**产品定位**：轻量级在线工具，即用即走

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽或点击上传，支持 JPG/PNG/WEBP | P0 |
| 自动处理 | 上传后自动调用 Remove.bg API 处理 | P0 |
| 结果预览 | 显示原图和处理后的对比 | P0 |
| 图片下载 | 一键下载透明背景PNG | P0 |

### 2.2 扩展功能（后期考虑）

- 批量处理
- 背景替换（换颜色/换图片）
- 历史记录

---

## 3. 技术方案

### 3.1 架构图

```
┌─────────────┐      ┌──────────────┐      ┌─────────────┐
│   用户前端   │ ──▶  │  后端 API    │ ──▶  │ Remove.bg   │
│  (HTML/JS)  │      │  (Node.js)  │      │    API      │
└─────────────┘      └──────────────┘      └─────────────┘
```

### 3.2 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML + Vanilla JS | 轻量，无框架依赖 |
| 后端 | Node.js | 处理图片上传和API转发 |
| API服务 | Remove.bg | 背景移除服务 |
| 托管 | 自有服务器 / Cloudflare Workers | 按需选择 |

### 3.3 接口设计

**请求**
```
POST /api/remove-bg
Content-Type: multipart/form-data

参数：image (图片文件)
```

**响应**
```
Content-Type: image/png

返回透明背景PNG图片
```
